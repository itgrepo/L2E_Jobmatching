📊 Use Case Diagram - ระบบ Job Matching

แสดงความสัมพันธ์ระหว่าง Actors และ Use Cases ในระบบจับคู่งาน

🔍 คำอธิบาย:

Actors: ผู้ใช้ของระบบ (Candidate, Employer, Admin)
Use Cases: การกระทำที่ผู้ใช้สามารถทำได้
System: ขอบเขตของระบบ

👨‍💼 Candidate (ผู้สมัคร)

  • สมัครสมาชิก
  • ตั้งค่าโปรไฟล์
  • ค้นหางาน
  • สมัครงาน
  • ดูสถานะใบสมัคร
  • รับคำแนะนำงาน
  • บันทึกงานโปรด

🏢 Employer (ผู้จ้างงาน)

  • สมัครสมาชิก
  • ลงประกาศงาน
  • แก้ไขประกาศ
  • ค้นหาผู้สมัคร
  • ดูใบสมัคร
  • เลือกผู้สมัครที่มีศักยภาพ
  • ติดต่อผู้สมัคร

👨‍💻 Admin (ผู้ดูแลระบบ)

  • จัดการผู้ใช้
  • จัดการงาน
  • สร้างรายงาน
  • ตรวจสอบข้อมูล
  • จัดการสิทธิ์
  • บำรุงรักษาระบบ
