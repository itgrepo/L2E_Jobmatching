🧠 AI Model (BERT)

การประมวลผล

แปลงข้อความ → Vector 384 มิติ
คำนวณ Cosine Similarity
ผลลัพธ์ = % Match (0–100)
สูตรคำนวณ:
similarity = (A·B) / (|A|×|B|)
ผลลัพธ์ × 100 = % Match
Skill Breakdown = เปรียบเทียบ List ตรงๆ

⚠️ ทำไมไม่ใช้ Mock JSON เดิม?

🔄 Training Pipeline

1

Load CSV

โหลด TBL_LEARNER + SKILL + EXPERINCE + job_post + job_candidate

2

สร้าง Profile Text

รวม about_me + skills + experience → text เดียว

3

สร้าง Training Pairs

hired → (learner_text, job_text, label=1.0) | rejected → label=0.0

4

Fine-tune BERT

paraphrase-multilingual-MiniLM-L12-v2 + CosineSimilarityLoss

5

บันทึก + วัดผล

save models/fine_tuned_bert → วัด Hit Rate@5, Precision@K

📐 สูตร % Match ที่แสดงใน UI

Learner Profile Vector
A
"Backend, 3ปี, สนใจ IT"
Job Description Vector
B
"Software Dev, Backend, Docker"
% Match Score
cos(A,B) × 100
= 80%
